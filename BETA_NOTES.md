# Finance Tracker — Beta Notes

## Overview
Finance Tracker is a local-first personal finance app for tracking transactions, budgets, and insights.

## What to Test
- Basic navigation across Home / Transactions / Budgets / More
- Adding transactions and verifying they appear in lists and totals
- Budgets creation and calculations
- Scheduled / recurring flows (where applicable)
- Data export (share/export flows)

## Known Limitations (Beta)
- AI features are under development and may be incomplete/experimental.
- Cloud sync is under development; this beta is primarily local-first.
- Release builds are currently debug-signed unless a release keystore is configured.

## Installing on Android (APK)
1. Download the provided `app-release.apk` onto your Android device.
2. Open the file and allow installation from unknown sources when prompted.
3. Install and launch “Finance Tracker”.

## Installing on Android (Play Console / AAB)
- If you are distributing via Play Console (Internal testing / Closed testing), upload the provided `app-release.aab`.

## Reporting Issues
When reporting issues, please include:
- Device model + Android version
- Steps to reproduce
- Screenshots (if applicable)
- Any visible error message
